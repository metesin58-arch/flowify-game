
# Ses Dosyaları

Bu klasör oyunun ses efektlerini barındırır. Lütfen aşağıdaki .mp3 dosyalarını bu klasöre yükleyin:

1. **click.mp3** - Buton tıklama sesleri için.
2. **tick.mp3** - Geri sayım (3..2..1) sırasındaki tık sesi.
3. **go.mp3** - Oyun başladığında çıkan ses (Düdük veya Gong).
4. **win.mp3** - Doğru cevap veya kazanma sesi.
5. **error.mp3** - Yanlış cevap veya hata sesi.

Not: Dosya isimlerinin tam olarak yukarıdaki gibi (küçük harflerle) olduğundan emin olun.
